Lưu ý*: 
- Responsive Trang:
    Sử dụng các class của Bootstrap để responsive đừng viết media query -> reponsive full thiết bị.
    Cài đặt extension trên gg chorme "Viewport Resizer"
- Chức Năng Trang:
    Đảm bảo trang web có đầy đủ các chức năng và hoạt động đúng logic.
- Ghi Đè Class Bootstrap:
    Sử dụng !important chỉ ghi đè các class của Bootstrap.
    Nếu không cần thiết, không sử dụng !important -> tránh ng sau code bị ghi đè.
- Quản Lý File và Code:
    t đã phân chia các file rõ ràng
    Đảm bảo code được viết đúng file tương ứng.
    Thêm comment (cmt) để giải thích cho người khác dễ đọc.
- Màu Sắc:
    Màu nền (background): #f3f3f9
    Màu chữ (text) sử dụng class Bootstrap: text-muted
- Icon:
    Sử dụng các icon từ Font Awesome và Box Icons.
